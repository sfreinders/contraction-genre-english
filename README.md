## Research Question
Are contractions more frequent in informal genres (Spoken, TV/Movies, Fiction) than formal genres (Academic, Newspapers)?

## Key Findings
- TV/Movies: 5.8M contractions
- Academic: 0.7M contractions 
- **Difference: 8x more** in informal genres
